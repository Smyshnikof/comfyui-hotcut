# ComfyUI · HOTCUT

**HOTCUT** добавляет в ComfyUI облачные модели изображений с оплатой в рублях —
огнями HOTCUT, без зарубежных карт и подписок.

Ключевое преимущество — **GPT Image 2**: чёткий текст, кириллица и лица, которые
локальным моделям (Flux, Qwen, SDXL) даются с трудом. Плюс редактирование
изображений и удаление фона — всё внутри привычного графа ComfyUI.

## Ноды

| Нода | Что делает |
|------|------------|
| **HOTCUT API** | Ваш ключ и адрес сервиса. Передаёт `config` остальным нодам и показывает баланс огней. |
| **HOTCUT · GPT Image 2** | Генерация по тексту (text-to-image): промпт, разрешение (1K/2K/4K), соотношение сторон → `IMAGE`. |
| **HOTCUT · GPT Image 2 Edit** | Редактирование по изображению (image-to-image): подаёте `IMAGE`, меняете или дорисовываете его в облаке → `IMAGE`. Легко сочетается с локальными нодами в одном графе. |
| **HOTCUT · Удалить фон** | Подаёте `IMAGE` — получаете изображение без фона (PNG с прозрачностью). ~10 огней. |

У ноды **HOTCUT API** есть кнопка **«Проверить баланс 🔥»** — показывает остаток огней без запуска графа.

## Установка

**Через ComfyUI Manager:** найдите «HOTCUT» и нажмите Install.

**Вручную:**
```bash
cd ComfyUI/custom_nodes
git clone https://github.com/Smyshnikof/comfyui-hotcut.git
cd comfyui-hotcut
pip install -r requirements.txt
```
Перезапустите ComfyUI — ноды появятся в категории **HOTCUT**.

## Ключ

1. Зарегистрируйтесь на [hotcut.ru](https://hotcut.ru) и пополните баланс огней.
2. **Студия → API → Создать ключ.** Скопируйте `hc_live_...` (показывается один раз).
3. Вставьте ключ в ноду **HOTCUT API**.

Чтобы ключ не попал в сохранённый `workflow.json`, оставьте в поле `api_key`
значение `$ENV` и задайте переменную окружения перед запуском ComfyUI:
```bash
export HOTCUT_API_KEY=hc_live_xxx   # Windows: setx HOTCUT_API_KEY hc_live_xxx
```

## Стоимость

Списывается огнями HOTCUT. Для GPT Image 2 зависит только от разрешения:
**1K ≈ 39**, **2K ≈ 65**, **4K ≈ 103** огня за кадр; удаление фона — около 10.
Соотношение сторон на цену не влияет. Перед запуском нода показывает, сколько спишется.

Списание — строго по факту успешной генерации. Если запрос не прошёл (ошибка
сервиса или блокировка модерацией), огни не списываются.

## Внешний вид

Ноды оформлены в фирменном стиле HOTCUT: тёмный корпус, логотип в заголовке и
превью стоимости «≈ N 🔥» на самой ноде. Реализовано лёгким фронт-расширением
(`web/hotcut.js`) на стабильных хуках ComfyUI — без вмешательства в граф, чтобы не
ломаться на обновлениях.
